# İstanbul Emlak Fiyat Tahminleme & Fırsat Dedektörü (2026)

Bu proje, İstanbul'daki güncel konut ilanlarını analiz ederek makine öğrenmesi ile fiyat tahmini yapar ve piyasa değerinin altındaki "fırsat" ilanları tespit eder.

## Kurulum
1. Bu repoyu clone'layın.
2. `data/` klasörüne Kaggle'dan indirdiğiniz veri setini ekleyin.
3. Gerekli kütüphaneleri kurun: `pip install -r requirements.txt`
4. Uygulamayı başlatın: `streamlit run app/app.py`

## Özellikler
- **Evin Değerini Öğren:** Form üzerinden ev özelliklerini girerek fiyat tahmini al.
- **Fırsat Dedektörü:** Piyasa fiyatının %20 ve daha altında kalan ilanları listele.
- **Kredi Faiz Oranı ve Taksit Simülasyonu:** kredi ve taksit tutarını hesapla.
